A Sinatra Project

  1. Make a folder - _sinatraapp
  2. Change to that directory - _cd sinatraapp
  3. Create file named - Gemfile

The Gemfile

Gemfile
source "https://rubygems.org"

gem 'sinatra'
gem 'haml'

$ bundle install

Create app.rb

app.rb
require 'sinatra'

get '/' do
  "Hello World"
end

Run your app

  • $ ruby app.rb
  • go to http://localhost:3000
